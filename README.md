Приложение размещается на сайте, его адресс:
https://the-witcher-pixel-hunt.netlify.app/

Приложение преднозначено для персональных компьютеров и запускается в браузерах, таких как Chromе и Firefox для корректной работы, необходимо: 
1. Иметь нормальный интернет интернет. (не менее 2 МБ/c) 
2. Иметь актуальную версию браузера. 
3. Также рекомендуемое разрешение экрана пользователя 16:9 или 4:3.

//Требования к составу и параметрам технических средств//

Компьютер пользователя, включающий в себя:
1. Процессор x64/x86 с тактовой частотой, не менее 1 ГГц;. 
2. Оперативную память объемом, не менее 2 Гб. 
3. Видеокарту, монитор, клавиатуру. 
4. Установелнный один из рекомендованных браузеров:
  (a) Firefox версией не ниже 72.0. 
  (b) Chrome версией не неже 79.0.
